Young Thug - Look Mama I Made It

‘Look Mama I Made It’ is a concept album cover which I designed for rapper Young Thug. I wanted to create something more on the simpler side for this one as well as include a few doodles to the artwork so that it wouldn’t be as boring and give it more of a grungy/graffiti type vibe to it.
